# NOWON-FIT

노원구 19개 동 대상 청년 이탈 조기경보 · 정책 처방 · 양면 추천 통합 시스템 (Streamlit)

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

브라우저에서 `http://localhost:8501` 로 접속하면 됩니다.

## 필요 파일 (전부 같은 폴더에 있어야 함)

```
app.py
requirements.txt
nowon_population.csv          # 총인구 (실데이터)
nowon_avg_age.csv             # 평균연령 (실데이터)
nowon_youth_population.csv    # 청년/청소년/아동 인구 (실데이터)
nowon_shelters.csv            # 비상대피시설 (실데이터)
nowon_housing_price.csv       # 부동산 실거래가 (실데이터)
nowon_welfare.csv             # 복지시설 집계 (실데이터)
```

## 구성

- **조기경보**: **청년인구 실데이터(2023.01~2025.12 월간)**를 선형회귀로 학습해 2027년 말까지 예측 —
  고위험/주의/안정 자동 분류. 보조 지표로 총인구·평균연령 10년 추이(2016-2025)도 함께 제공.
- **정책 처방**: 동의 YSI(청년 정착 지수) 취약 요소와 정책별 비용·효과를 바탕으로 예산 제약 하
  최적 정책 조합을 그리디 알고리즘으로 계산.
- **양면 추천**: 청년 모드(우선순위 슬라이더 → 맞춤 동 추천 Top3) / 행정 모드(동 선택 → 취약 요소
  및 개선 효과 시뮬레이션).

## 📊 데이터 현황 (2026-07 기준)

| 데이터 | 상태 | 출처 | granularity |
|---|---|---|---|
| 청년/청소년/아동 인구 (2023.01-2025.12, 월간) | ✅ **실데이터** | 행안부 주민등록 인구 기타현황 | 행정동 (정확) |
| 총인구 (2016-2025, 연간) | ✅ **실데이터** | 행안부 주민등록 인구 및 세대현황 | 행정동 (정확) |
| 평균연령 (2016-2025, 연간) | ✅ **실데이터** | 행안부 주민등록 인구 기타현황(평균연령) | 행정동 (정확) |
| 부동산 실거래가 (아파트) | ✅ **실데이터** | 서울시 부동산 실거래가 정보 | 법정동 → 행정동 균등 배분 추정 |
| 비상대피시설 (수/면적) | ✅ **실데이터** | 노원구 비상대피시설 목록 | 행정동 (정확) |
| 복지시설 4종 (노인주거·노인의료·장애인재활·다문화가족) | ✅ **실데이터** | 서울시 사회복지시설 목록 | 법정동 → 행정동 균등 배분 추정 |
| 상가/생활 인프라 밀도 | ⚠️ 시뮬레이션 | 소상공인시장진흥공단 상가정보 API(인증키 필요, 미연동) | - |
| 통근시간 | ⚠️ 시뮬레이션 | 미확보 | - |

**법정동 → 행정동 배분 관련 주의**: 실거래가와 복지시설 원본 데이터는 주소가 법정동
(예: '상계동')까지만 있어, 그 법정동에 속한 여러 행정동(예: 상계1~10동)에 평균값을
동일하게 적용했습니다. 행정동 간 미세한 차이는 반영되지 않은 근사치입니다. 정확한
행정동 단위 데이터를 구하면 `nowon_housing_price.csv`, `nowon_welfare.csv`를
교체하면 됩니다.

**범죄/치안 데이터**: 서울시가 공개하는 범죄 통계는 경찰서 관할(구 단위) 기준으로만
집계되어 동 단위 데이터가 존재하지 않습니다. 이 앱에서는 대신 실제 동별 데이터가 있는
"비상대피시설(재난 안전 인프라)"을 안전 지표로 사용합니다.

## 원본 데이터 파일 (참고용, 앱 실행에는 불필요)

앱은 전처리된 CSV만 사용합니다. 원본 엑셀/CSV를 다시 처리하고 싶다면 아래 파일 구조를 참고하세요.

- `201612_202512_주민등록인구및세대현황_연간.xlsx` → `nowon_population.csv`
- `201612_202512_주민등록인구기타현황_평균연령__연간.xlsx` → `nowon_avg_age.csv`
- `202301_202512_주민등록_인구_기타현황_아동청소년청년_인구현황__월간.xlsx` → `nowon_youth_population.csv`
- `20260709_비상대피시설목록.xlsx` → `nowon_shelters.csv`
- `서울시_부동산_실거래가_정보.csv` → `nowon_housing_price.csv`
- `서울시_사회복지시설_*.csv` (4종) → `nowon_welfare.csv`
