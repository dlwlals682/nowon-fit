# NOWON-FIT

노원구 19개 동 대상 청년 이탈 조기경보 · 정책 처방 · 양면 추천 통합 시스템 (Streamlit 데모)

## 실행 방법

```bash
pip install -r requirements.txt
streamlit run app.py
```

브라우저에서 `http://localhost:8501` 로 접속하면 됩니다.

## 구성

- **조기경보**: 동별 청년인구 시계열을 선형 추세로 학습해 2027년까지 예측, 위험도(고위험/주의/안정) 표시
- **정책 처방**: 동의 YSI(청년 정착 지수) 취약 요소와 정책별 비용·효과를 바탕으로 예산 제약 하 최적 정책 조합을 그리디 알고리즘으로 계산
- **양면 추천**: 청년 모드(우선순위 슬라이더 → 맞춤 동 추천 Top3) / 행정 모드(동 선택 → 취약 요소 및 개선 효과 시뮬레이션)

## ⚠️ 데이터 안내

현재 버전은 다음 공공데이터 소스의 통계적 특성을 반영한 **시뮬레이션 데이터**로 동작합니다.
실 서비스로 전환하려면 `app.py`의 `load_data()` 함수 내부를 아래 실제 API 호출로 교체하세요.

| 데이터 | 출처 | 용도 |
|---|---|---|
| 동별 청년 인구 | 행정안전부 주민등록통계 | 조기경보 시계열 |
| 실거래가 | 국토교통부 실거래가 API | 주거비 지수 |
| 상가 정보 | 소상공인진흥공단 상가(상권)정보 | 인프라 밀도 |
| 범죄 통계 | 서울 열린데이터광장 | 안전 지수 |
| 복지시설 | 노원구 열린데이터광장 | 복지 인프라 |

## 커스터마이징 포인트

- `POLICY_OPTIONS`: 정책 수단별 단가/효과 딕셔너리 — 실제 노원구 예산 단가로 교체 가능
- `compute_ysi()`: YSI 가중치(주거 30% / 인프라 25% / 안전 25% / 통근 20%) 조정 가능
- `forecast_population()`: 현재 선형회귀(1차) 대신 ARIMA 등 정교한 시계열 모델로 교체 가능
